01

Site Name

KeyStart
Beginner's Guide to Mechanical Keyboards

KeyStart combines keyboard with start — speaking directly to someone taking their first steps into the mechanical keyboard hobby. The name is short, easy to remember, and implies guidance without being condescending. It doesn't require prior knowledge to understand, which matches the site's core audience.

Optional domain keystart.guide
02

Site Purpose

KeyStart is an educational resource for people who are curious about mechanical keyboards but don't know where to begin. The site covers what mechanical keyboards are and why people care about them, the three main switch types (linear, tactile, clicky) and what each one feels like, the key components of a keyboard (PCB, case, switches, keycaps, stabilizers), a budget-based buying guide for first-time builders or buyers, a side-by-side comparison of popular entry-level options, and a glossary of common hobby terms.

Content is structured so a visitor can read front-to-back or jump to any topic independently. JavaScript will be used to add interactivity — a switch type comparison tool, glossary filtering, and a dynamic keyboard recommendation based on user input.

04

Color Schema

The palette is inspired by custom keycap sets — deep, neutral base colors with a sharp accent that commands attention without overpowering.

#0D0D0D Near Black

Page background. Creates the dark, focused atmosphere of a late-night build session. Used on <body>.

#1A1A1A Surface

Card and section backgrounds. Lifts content slightly off the page background. Used on .card, .nav.

#E8FF47 Acid Yellow — Primary Accent

The single bold color. Used on headings, key labels, hover states, links, and the logo. Inspired by the iconic "Varmilo" and "GMK" keycap colorways.

#F0F0F0 Off White — Body Text

Primary readable text. Slightly off-white reduces contrast fatigue on dark backgrounds for long-form reading.

#888888 Muted — Secondary Text

Labels, captions, metadata, section numbers, and placeholder content. Keeps hierarchy clear without adding more colors.

#2D2D2D Border / Divider

Card borders, horizontal rules, and subtle separation between elements. Keeps the layout structured without heavy lines.

05

Typography

Two typefaces — chosen for contrast in personality while sharing the same clean, geometric DNA.

Space Grotesk
Space Grotesk — Display & Headings

Used for: h1, h2, h3, navigation items, the logo, and the site name. Its slightly quirky letterforms (note the curved terminals on 'a', 'r', 't') give the technical content a personality without being unreadable. Weight: 700 for headings, 500 for subheadings.

Inter — clean & readable for long text
Inter — Body Text & UI

Used for: paragraphs, list items, card descriptions, table content, form labels, and captions. Highly legible at small sizes and optimized for screens. Weight: 400 for body copy, 500 for labels and small UI elements.

monospace → hotswap, PCB, RGB
System Monospace — Code & Technical Terms

Used for: glossary terms, technical specifications, part names in context, and any code snippets. Uses the browser's default monospace stack (ui-monospace, 'Courier New') — no additional font load needed.
